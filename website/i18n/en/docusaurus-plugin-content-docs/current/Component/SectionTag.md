断面情報の表示を行うコンポーネントのカテゴリ

## Column Section Tag

![](../../images/Component/ColumnSectionTag.png)

柱の断面情報表示

|入力|説明|
|---|:---:|
|Data|Load STB file コンポーネントの Data 出力を入力|
|Size|Rhino のビューポートに表示する文字サイズ。デフォルトは12です|

|出力|説明|
|---|:---:|
|SecTag|符号のリスト|
