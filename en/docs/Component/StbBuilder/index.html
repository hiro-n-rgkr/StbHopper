StbBuilder

Categories of components that create information for STB export

note

The usage and notes for this component are also described in Export ST-Bridge file for more details.

Frame Builder by angleâ€‹

Create the frame information from the Karamba3D model.
To classify columns and beams, enter the angle of the global coordinate with respect to the Z-axis in Angle, and if the angle is within that range, the column is determined to be a column.

Inputdescription
ModelInput the Model output of Karamba3D
AngleAngle (rad) to be used when determining between column and beam
outputdescription
MemberMember information (StbMembers)
SectionSection information (StbSections)
